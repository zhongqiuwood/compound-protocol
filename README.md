# Requirement
=================
## erc20: hToken(h币)
* 等同于compound的cToken

## erc20: huo(o币，承担comp币的治理功能)
* 80% 预挖
* 剩余20%每挖1个i币, 奖励0.1个o币
* 上线20周后就不再奖励，没有挖完就销毁

## erc20: hubi(i币)
* 挖矿方式和comp一样, 按照分发comp的方式(根据借贷提供的流动性多少)
* 总量8亿，每4年减半
* 最小单位是1
